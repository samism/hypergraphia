# Running Drills

A short warmup before any run, after the first easy half-mile.

A-skip and B-skip down the block, twenty meters each. High-knees with quick turnover. Butt kicks. Carioca side-to-side. Strides — four reps of about a hundred meters at 5K pace, full recovery between. The point is to wake up the running form, not to add training load. Total time: five minutes.

If the run is a workout (intervals, tempo), skip the carioca and add an extra stride. If the run is recovery, skip the strides entirely.
