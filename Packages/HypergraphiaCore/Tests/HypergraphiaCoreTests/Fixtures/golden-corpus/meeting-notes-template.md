# Meeting Notes Template

The structure I use for any meeting that's worth taking notes on.

```
# {Meeting Title} — {Date}

**Attendees:** ...
**Goal:** ... (one sentence; if I can't write it, the meeting wasn't ready)

## What we decided

- ...

## Open questions

- ...

## Action items

- [ ] {who} — {what} — {by when}
```

The two parts that actually matter: "what we decided" and "action items." If neither has anything in it after the meeting, the meeting was a discussion, not a working session, and that's fine — but flag it so we don't accidentally count it as progress.

Action items always have an owner, an action, and a due date. "Someone should look at this" is not an action item; it's a wish.
